## Формат конфиг-файла:

```ini
[Flask]
// Ключ-конфига-для-Flask = значение-конфига-для-Flask
// Пример:
SECRET_KEY = ....
SQLALCHEMY_DATABASE_URI = sqlite:///database.db
SQLALCHEMY_TRACK_MODIFICATIONS = False

[RunConfig]
host = ...
port = ...
debug = ...
```
